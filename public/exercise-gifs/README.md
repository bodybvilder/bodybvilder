# Exercise GIFs

Letakkan file GIF hasil generate di sini.

## Naming Convention
- Format: `[exercise-id].gif`
- Contoh: `pushup.gif`, `barbell-squat.gif`, `pose-front-double-biceps.gif`

## Spesifikasi
- Ukuran: 300x300px
- Background: transparan atau #0a0a0a
- Warna figure: #e0e0e0
- Aksen: #C8FF00 (lime green)
- Format: looping animated GIF
- FPS: 12-24fps

## Daftar File yang Dibutuhkan
Lihat GIF_GENERATION_GUIDE.md di root project untuk semua 81 prompts.
