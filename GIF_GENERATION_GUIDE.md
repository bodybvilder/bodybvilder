# BODYBVILDER — Panduan Generate GIF Exercise & Pose

## Spesifikasi GIF (WAJIB DIIKUTI)

| Parameter | Nilai |
|-----------|-------|
| Ukuran | **300x300px** (atau 200x200px) |
| Background | **Transparan** atau `#0a0a0a` (hitam hampir pekat) |
| Warna figure | `#e0e0e0` (abu terang/putih) |
| Warna aksen | `#C8FF00` (lime green — untuk highlight sendi, jalur gerak) |
| Format | **Looping animated GIF** |
| FPS | **12–24 fps** |
| Durasi loop | **1.5–3 detik** |
| Gaya | **Minimalist flat vector** — BUKAN foto realistik, BUKAN 3D render |
| Nama file | `[exercise-id].gif` (contoh: `pushup.gif`, `barbell-squat.gif`) |

---

## Cara Pakai dengan Kimi AI / Claude AI

1. **Copy satu prompt** dari daftar di bawah
2. **Paste ke Kimi/Claude** dengan instruksi:
   > "Buat animated GIF dari deskripsi ini sesuai spesifikasi: 300x300px, background transparan/#0a0a0a, figure warna #e0e0e0, aksen lime green #C8FF00, minimalist flat vector, looping 12-24fps, 1.5-3 detik per loop"
3. **Simpan hasilnya** ke folder: `public/exercise-gifs/[nama-file].gif`
4. Ulangi untuk setiap exercise

## Tips Batch Processing

Untuk efisiensi, kirim **5–10 prompt sekaligus** ke Kimi/Claude dengan format:
> "Generate 5 animated GIFs sesuai spesifikasi berikut: [spesifikasi umum]. Untuk masing-masing: [prompt 1], [prompt 2], ..."

Kimi AI mendukung image generation termasuk GIF. Claude bisa dipakai untuk membuat SVG animated atau mengarahkan ke tools lain.
